## Development instructions
### JavaScript
- Install dependencies
```sh
npm ci
```
- Run tests
```sh
# Run all tests
npm test
# Run tests in a subfolder
npm test 2022/
```
- Run solution against puzzle input
```sh
node <path to index.js file>
# Eg
node node 2021/02-dive/index.js
```
